Blue Badge rights and responsibilities

The way a Blue Badge can be used differs between England, Scotland and Wales:

  • for England, you can read the rules on GOV.UK
  • for Scotland, you can read the rules on mygov.scot
  • for Wales, you can read the rules on gov.wales

Once you’ve read the rules for using a Blue Badge in the nation you’ll be using it in you should be ready to apply.
